PalPalooza App

The Pal-Palooza App is a serverless, progressive web application (PWA) built with Vue.js. It's designed to connect individuals based on shared interests, whether they're traveling, looking for a sports partner, or just want to grab a drink. The app integrates seamlessly with Firebase for backend services and FirebaseUI for user authentication, offering a blend of real-time chat functionality, user profile management, and activity posting and filtering.

Technologies Used:

Vue.js, Firebase, FirebaseUI, Bootstrap 5, Real-time Chat Functionality, Single Page App (SPA) Techniques
